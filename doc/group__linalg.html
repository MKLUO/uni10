template<typename UniType >
std::vector< Matrix< UniType > > uni10::Ldq ( const Block< UniType > &  kblk)

Compute the LQ decomposition of a matrix.

Factor the matrix kblk as l*d*q, where l is lower-triangular, d is diagonal and q is orthonormal.

Parameters
[in]kblkMatrix to be factored.
Returns
Vector of result matrices. The elements are (in sequence): l The lower-triangular matrix, d The diagonal matrix. q A matrix with orthonoraml rows. l*d is equivalent to the lower triangular matrix in the conventional LQ decomposition.

Compute the QR decomposition of a matrix.

Factor the matrix kblk as q*d*r, where q is orthonormal, d is diagonal and r is upper-triangular.

Parameters
[in]kblkMatrix to be factored.
Returns
Vector of result matrices. The elements are (in sequence): q A matrix with orthonoraml columns, d The diagonal matrix. r The upper-triangular matrix.d*r is equivalent to the upper triangular matrix in the conventional QR decomposition.
