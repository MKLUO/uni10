Driver function for High-order SVD.

Performs High order SVD of UniTensor.

Parameters
[in]group_labelsOrdered labels of the bonds
[in]groupsNumber of external bonds in each mode
[in]groupSizeNumber of modes
[out]LsSingular values on each direction
***** Example ******
____________ ______ ____
| | 0___| U1 |__-1 ___ __-2___| |___1
8___| |___0 |______| _|___|_ | U2 |___2
| | | | |____|
9___| |___1 __11_| |
| | __3__| |
11__| |___2 | S |
| | ______ | | ____
6___| |___3 8__| |__-4__| |__-3_| |___4
| | 9__| U4 | |_______| | U3 |___5
5___| |___4 |______| |____|___6
| |
|____________|

We want to decompose the tensor into a tensor of four modes and two bonds unchanged.

Mode 1 {0,-1}
Mode 2 {1,2,-2}
Mode 3 {4,5,6,-3}
Mode 4 {8,9,-4}
Fixed {3,11}

The input parameters are

group_labels = { 0, 1, 2, 4, 5, 6, 8, 9, 3, 11 }
groups={1,2,3,2}
groupSize=4

Returns unitaries U1, U2, U3, U4, and core tensor S

Get the optimal contrction order obtain by PreConstruct or Launch. For example, the following Network 

A: 1 2; 3 4
B: 3 4; 5 6
C: -1; 7 1
D: -2; 2 8
E: 7 5; -3
F: 6 8 -4
TOUT: -1 -2; -3 -4

with all bonds have dimension 2 will yield the contraction order below: 

L1N0
(16) __________
  |           |
  |           |
L0N1'         |
(16) ____     |
  |     |     |
  |     |     |
L0N2   L0N1  L0N0
(16)   (16)  (16)

L0N0       L0N1       L0N2
(16)____   (16)____   (16)____
 |     |    |     |    |     |
 |     |    |     |    |     |
 E     C    F     D    B     A
(8)   (8)  (8)   (8)  (16)  (16)
Note
The optimal contraction order may vary for same Network file with different tensors put in it.
Returns
A string indicates the optimal contraction order.
